# JSON行分割マージツール

複数のJSONファイルを読み込み、1行に1つのデータ（オブジェクト）が入る形式で1つのファイルに結合するツールです。

## 特徴

- **簡単なGUI操作**: フォルダを選択するだけで、中にあるすべてのJSONファイルをスキャンします。
- **改行区切り形式（JSONL相当）**: 複数のJSONファイルを、1行ごとに独立したデータとして出力します。
- **リスト・単体両対応**: 元のJSONがリスト形式 `[]` でも、単体の辞書形式 `{}` でも適切に展開して結合します。

## 使い方

1. `JSON_Gatchanko_Tool.exe` を実行します。
2. 「フォルダを選択」ボタンを押し、JSONファイルが保存されているフォルダを指定します。
3. 「改行区切りで結合開始！」ボタンを押します。
4. 選択したフォルダ内に `combined_lines.json` が生成されます。

## 動作環境

- Windows 10 / 11
